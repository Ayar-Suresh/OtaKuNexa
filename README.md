## 📱 About OtaKuNexa

OtaKuNexa is a feature-rich Flutter application designed for anime lovers. It brings thousands of anime series directly to your fingertips, integrating beautifully with the YouTube API and Telegram to provide an endless stream of content in a seamless, high-quality user interface.

## ✨ Features

- 🎥 **Massive Library:** Access to 1000s of anime episodes.
- 📺 **YouTube API Integration:** Watch high-quality streams directly inside the app using a native player.
- ✈️ **Telegram Integration:** Get the latest updates, links, and community features right within the app.
- 🚀 **Smooth & Modern UI:** Beautifully crafted using Flutter with stunning animations and slick design elements.
- 🔔 **Firebase Powered:** Integrated with Firebase Cloud Firestore & Remote Config for real-time updates and seamless data fetching.
- 🎨 **Customizable:** Experience the app with a beautiful user-friendly interface.

## 🛠️ Technology Stack

- **Framework:** [Flutter](https://flutter.dev/)
- **UI & Animations:** `shimmer`, `cached_network_image`, `animate_do`, `lottie`
- **Video Playback:** `chewie`, `media_kit`, `youtube_player_flutter`
- **Backend Services:** Firebase Firestore, Firebase Remote Config
- **Networking:** `http`, `webview_flutter`, `url_launcher`
